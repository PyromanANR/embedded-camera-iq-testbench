Troubleshooting

  • If the dashboard says the device is offline, start the C++ Camera Gateway on port 8081.
  • If a port is busy, close the previous uvicorn process or run on another port.
  • If no photos exist in stock, the generated placeholder keeps the system runnable.
  • If OpenCV import fails, reinstall dependencies in the selected PyCharm interpreter.
